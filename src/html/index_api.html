Accounts

Request

Mainnet : https://thecelo.com/api/v0.1?method=accounts&page=[page]&count=[count]&sort_flg=[sort_flg]

Testnet Baklava : http://baklava.thecelo.com/api/v0.1?method=accounts&page=[page]&count=[count]&sort_flg=[flg]

Request

  • page: page number, starting from 0
  • count: the number of items displayed per page
  • sort_flg: sort flag (0: celo, 1: cusd, 2: txs)

Response

  • CELO_totalSupply
  • cUSD_totalSupply
  • totalCELO
  • totalcUSD
  • CELO_total_addresses
  • cusd_total_addresses
  • cusd_total_transfers
  • accounts{}
    • key : address
    • value : [0-n]
      • 0 : TXs
      • 1 : CELO
      • 2 : cUSD
      • 3 : Name
      • 4 : lockedCELO
      • 5 : nonVotingLockedCELO
      • 6 : pendingWithdrawals
      • 7 : metadataURL
